
# 🧠 Python for AI & Data Science

## 📌 Course Modules Overview

| Module | Topic Area                  | Description                                                                 |
|--------|-----------------------------|-----------------------------------------------------------------------------|
| 1      | Programming Fundamentals    | Learn Python basics including data types, expressions, and string handling. |
| 2      | Python Data Structures      | Explore core structures like lists, sets, tuples, and dictionaries.         |
| 3      | Control Flow & Functions    | Understand conditional logic, loops, functions, scope, and exception handling. |
| 4      | Functional Programming & OOP| Dive into higher-order functions, decorators, object-oriented principles, and method types. |
| 5      | Working with Data           | Learn to read/write files, use NumPy for arrays, pandas for dataframes, and handle various file formats. |
| 6      | APIs & Web Data Collection  | Work with REST APIs, HTTP requests, and perform basic web scraping with BeautifulSoup. |
| 7      | Advanced API Integration    | Build and test APIs using FastAPI, OpenAI, Gemini APIs, and learn routing, parameters, and HTTP methods. |
| 8      | Capstone Project            | Apply all skills in a real-world Python mini-project for end-to-end practice. |


## 🔗 How to Run the Notebooks

You can run any notebook interactively using Google Colab:

1. Click on any `.ipynb` file in this repository.
2. Then click the **"Open in Colab"** button at the top of the file preview on GitHub.

📌 *Direct Colab badges for each notebook will be added soon.*


## 🛠️ Requirements

Most notebooks are designed to run smoothly in **Google Colab**, with `!pip install` commands included inside each file where needed.  
So you **don’t need to install anything manually** unless you're running locally.

For full local execution (optional), install these:

```bash
pip install numpy pandas requests fastapi uvicorn openai beautifulsoup4
```
